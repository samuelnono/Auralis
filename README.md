# Auralis

Auralis is a frequency- and emotion-aware music recommendation system.

## Roadmap
### Phase 0 — Setup ✅
- Git + GitHub repo
- Python + virtual environment
- requirements.txt tracked

### Phase 1 — Core Pipeline (next)
1) Extract audio features (frequency + spectral features)
2) Map features to emotion signals
3) Recommend songs based on emotion + similarity
